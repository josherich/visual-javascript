## Sample
[demo](https://zyl11.csb.app/) for the following sample code
```
var a = 42;
var b = 5;
function addA(d) {
  return a + d;
}
c = addA(2) + b;
if (c + b > 0) {
  a = 0;
} else {
  a = 1;
}
```
